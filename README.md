## 🚀 Cómo Usar

### Opción 1: Abrir directamente en el navegador
```bash
# Simplemente haz doble clic en index.html
```

### Opción 2: Usar un servidor local (Python)
```bash
python -m http.server 8000
# Luego abre http://localhost:8000 en tu navegador
```

### Opción 3: Usar Live Server (VS Code)
1. Instala la extensión "Live Server"
2. Click derecho en `index.html`
3. Selecciona "Open with Live Server"

## 🎯 Personalización

### 1. Cambiar Logo
En `index.html`, reemplaza:
```html
<div class="logo">Logo</div>
```
Por tu nombre o logo:
```html
<div class="logo">Cristian Diaz</div>
```

### 2. Cambiar Colores
En `styles.css`, modifica las variables CSS:
```css
:root {
    --primary-color: #0066ff;      /* Azul principal */
    --secondary-color: #333333;    /* Gris oscuro */
    --light-bg: #f5f5f5;           /* Fondo claro */
    /* ... más variables */
}
```

### 3. Agregar Imágenes
Reemplaza los placeholders grises por imágenes reales:
```html
<!-- Antes -->
<div class="placeholder">Imagen</div>

<!-- Después -->
<img src="ruta/a/tu/imagen.jpg" alt="Descripción">
```

### 4. Actualizar Información de Contacto
En `index.html`, sección `#contacto`:
```html
<a href="mailto:tu-email@ejemplo.com">tu-email@ejemplo.com</a>
<a href="tel:+57XXXXXXXXX">+57 XXX XXX XXXX</a>
```

### 5. Agregar Redes Sociales
Actualiza los links en la sección de contacto:
```html
<a href="https://linkedin.com/in/tu-perfil" class="social-link" title="LinkedIn">in</a>
<a href="https://github.com/tu-usuario" class="social-link" title="GitHub">⚙</a>
```
